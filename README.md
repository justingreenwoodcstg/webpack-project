# WebPack with TypeScript for a simple Game
## Notes:
- Install NVM
- Install NPM `nvm install lts` `nvm use lts`
- Install yarn `npm install --global yarn`
- 
```
yarn init
yarn add webpack webpack-cli -D
npx webpack init
```
